Overview:
This project is a collaborative Animal & Pet Information Website created by students to learn the fundamentals of web development. It includes multiple informational pages, each created by different students to promote teamwork, creativity, and practical coding experience.The website is fully responsive using Bootstrap 5, with clean styling, simple layout, and easy navigation.
 Features:
 Multi-page website structure

Responsive design using Bootstrap

Simple & beginner-friendly UI

Individual pages created by different students

Organized folder structure
Easy to modify and expand
Pages Included
Page	Description
Home	Introduction to the website
Animals List	Common pets such as Dog, Cat, Rabbit, Parrot, etc.
Care Tips	Instructions for taking care of pets
Fun Facts	Interesting animal facts
Contact	Basic contact information

HTML5

CSS3
